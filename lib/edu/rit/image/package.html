Package edu.rit.image contains classes for creating image files. The following image file formats are supported:

  • Bilevel PNG -- class BilevelPngWriter
  • Grayscale PNG -- class GrayPngWriter
  • Indexed color PNG -- class IndexPngWriter
  • 24-bit color PNG -- class ColorPngWriter

Package edu.rit.image also contains classes for creating animated image files. The following animated image file formats are supported:

  • 24-bit color APNG -- class ColorApngWriter
