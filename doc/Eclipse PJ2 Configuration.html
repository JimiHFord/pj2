Eclipse PJ2 Configuration

This guide walks you through configuring the Eclipse IDE with the Parallel Java 2 (PJ2) framework.

Navigation:
  ↳ Overview
  ↳ Download PJ2
  ↳ Configure Eclipse

 
TRANSLATE

Overview

This guide is intended to show you how to configure the Eclipse IDE with the Parallel Java 2 (PJ2) framework created by Rochester Institute of technology Professor Alan Kaminsky.

Note: Parallel Java 2 (PJ2) is the work of Alan Kaminsky, and is in no way related to Shane-Hale.com. This guide is only intended to help others to configure this framework with Eclipse, any questions related to specific operations of PJ2 should be referenced at the PJ2 website.

Firstly, this guide assumes that you have already configured, and installed the Eclipse IDE, and Java Development Kit (JDK). Otherwise, these applications can be downloaded at their respective websites:

               Eclipse IDE: here
               JDK: here

Download PJ2

To download the PJ2 java archive (executable distribution) file from Alan Kaminsky’s webpage here.

For easier reference, you should rename this JAR file to “pj2.jar”

Configure Eclipse

Begin by creating a new Java Project (if you have not already done so)

Step 3)      Right click on the newly created project within the Eclipse Package Explorer, click ‘Build Path', then ‘Add External Archives…’

Step 4)      Select the (pj2.jar) file that was previously downloaded in step 1 from Alan Kaminsky’s website

Step 5)      Right click on the project in the Package Explorer, click ‘Run As’, then ‘Run Configurations…’

Step 6)      Go to the ‘Main’ tab, and type ‘pj2’ into the Main class textbox.

Step 7)      Now, click to the ‘Arguments’ tab, and type your project name into the ‘Arguments’ textbox.

Step 8)      Finally, click ‘Run’

Step 9)      The application should now execute, and the program output displayed in the console.
